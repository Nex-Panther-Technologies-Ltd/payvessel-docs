---
title: "Create Wallet"
description: "Guide: when and why to create a managed wallet"
---

Create a managed wallet when you need a **dedicated balance** for a specific purpose—for example settlements, refunds, partner funds, or regional balances.

Common patterns:

- Separate wallets for **operational funds**, **refunds**, and **partner balances**.
- One wallet per **business unit** or **country**.
- Dedicated wallets for **testing** vs **production** flows.

For the endpoint, request/response payloads, and code samples, see **API reference → Wallets → Create Wallet**.

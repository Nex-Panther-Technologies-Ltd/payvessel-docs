# Create Wallet

Create a digital wallet for customers to securely store funds, make payments, and receive money. PayVessel wallets support multiple currencies and provide real-time balance tracking.

<Note>
Digital wallets enable you to build comprehensive fintech solutions with account-to-account transfers, bill payments, and merchant transactions.
</Note>

## Endpoint

<Badge variant="success">**POST**</Badge> `/wallets/create`

## Headers

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `Authorization` | string | **Required** | Bearer token with your secret API key |
| `Content-Type` | string | **Required** | Must be `application/json` |
| `Idempotency-Key` | string | **Recommended** | Unique key to prevent duplicate wallet creation |

## Request Body

<ParamField body="customer_id" type="string" required>
  Unique identifier for the customer who will own this wallet
  
  <Note>One customer can have multiple wallets in different currencies</Note>
</ParamField>

<ParamField body="currency" type="string" default="NGN">
  Primary currency for the wallet
  
  **Supported currencies:** `NGN`, `USD`, `GBP`, `EUR`, `GHS`, `KES`, `ZAR`
</ParamField>

<ParamField body="wallet_type" type="string" default="personal">
  Type of wallet to create
  
  **Available types:**
  - `personal` - Individual customer wallet
  - `business` - Business/corporate wallet
  - `savings` - High-yield savings wallet
  - `current` - Current account wallet
</ParamField>

<ParamField body="label" type="string">
  Human-readable label for the wallet (e.g., "Main Account", "Savings", "Business Funds")
</ParamField>

<ParamField body="customer_info" type="object" required>
  Customer information for wallet creation
  
  <Expandable title="Customer Info Object">
    <ParamField body="customer_info.first_name" type="string" required>
      Customer's first name
    </ParamField>
    
    <ParamField body="customer_info.last_name" type="string" required>
      Customer's last name
    </ParamField>
    
    <ParamField body="customer_info.email" type="string" required>
      Customer's email address
    </ParamField>
    
    <ParamField body="customer_info.phone" type="string" required>
      Customer's phone number in international format
    </ParamField>
    
    <ParamField body="customer_info.date_of_birth" type="string">
      Customer's date of birth in YYYY-MM-DD format
    </ParamField>
    
    <ParamField body="customer_info.bvn" type="string">
      Customer's Bank Verification Number (required for NGN wallets)
    </ParamField>
    
    <ParamField body="customer_info.address" type="object">
      Customer's address information
      
      <Expandable title="Address Object">
        <ParamField body="customer_info.address.street" type="string">
          Street address
        </ParamField>
        
        <ParamField body="customer_info.address.city" type="string">
          City
        </ParamField>
        
        <ParamField body="customer_info.address.state" type="string">
          State or province
        </ParamField>
        
        <ParamField body="customer_info.address.country" type="string">
          Country code (ISO 3166-1 alpha-2)
        </ParamField>
        
        <ParamField body="customer_info.address.postal_code" type="string">
          Postal or ZIP code
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="initial_deposit" type="string" default="0">
  Initial deposit amount in the smallest currency unit (optional)
  
  <Warning>Initial deposits require sufficient balance in your main account</Warning>
</ParamField>

<ParamField body="features" type="object">
  Wallet features configuration
  
  <Expandable title="Features Object">
    <ParamField body="features.debit_card" type="boolean" default="false">
      Enable virtual debit card for the wallet
    </ParamField>
    
    <ParamField body="features.bill_payments" type="boolean" default="true">
      Allow bill payments from this wallet
    </ParamField>
    
    <ParamField body="features.transfers" type="boolean" default="true">
      Enable transfers to/from this wallet
    </ParamField>
    
    <ParamField body="features.savings_interest" type="boolean" default="false">
      Enable interest earning for savings wallets
    </ParamField>
    
    <ParamField body="features.overdraft" type="boolean" default="false">
      Allow overdraft facility (business wallets only)
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="limits" type="object">
  Transaction limits for the wallet
  
  <Expandable title="Limits Object">
    <ParamField body="limits.daily_transaction_limit" type="string">
      Maximum daily transaction amount in smallest currency unit
    </ParamField>
    
    <ParamField body="limits.monthly_transaction_limit" type="string">
      Maximum monthly transaction amount in smallest currency unit
    </ParamField>
    
    <ParamField body="limits.single_transaction_limit" type="string">
      Maximum single transaction amount in smallest currency unit
    </ParamField>
    
    <ParamField body="limits.daily_transaction_count" type="integer">
      Maximum number of transactions per day
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="metadata" type="object">
  Additional information about the wallet
  
  ```json
  {
    "source": "mobile_app",
    "referral_code": "REF123",
    "customer_segment": "premium",
    "created_by": "customer_service"
  }
  ```
</ParamField>

      await saveCustomerWallet(customerData.id, wallet.data);
      
      // Generate virtual debit card if enabled

      }
      
      return wallet.data;
    };

## Webhook Events

This endpoint triggers the following webhook events:

- `wallet.created` - Wallet successfully created
- `wallet.activated` - Wallet activated and ready for use
- `card.issued` - Virtual card issued (if debit_card feature enabled)
- `account.opened` - Virtual bank account opened (for NGN wallets)

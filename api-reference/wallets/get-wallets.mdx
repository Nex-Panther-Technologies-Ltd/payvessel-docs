---
title: "Get Wallets"
description: "List or retrieve managed wallets for your business"
---


**Retrieve all managed wallets** associated with your business. This is useful for monitoring balances across multiple wallets and reconciling transfers.

## Endpoint

<Badge variant="success">**GET**</Badge> `/pms/wallets`

## Headers

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `Authorization` | string | **Required** | Bearer token with your secret API key |

## Response

<ResponseField name="status" type="string">
  Request status - `"success"` or `"error"`.
</ResponseField>

<ResponseField name="data" type="array">
  <Expandable title="Wallet Object">
    <ResponseField name="id" type="string">
      Unique wallet ID.
    </ResponseField>
    <ResponseField name="name" type="string">
      Custom name or label for the wallet.
    </ResponseField>
    <ResponseField name="balance.available_balance" type="string">
      Available balance.
    </ResponseField>
    <ResponseField name="balance.ledger_balance" type="string">
      Ledger balance.
    </ResponseField>
    <ResponseField name="currency" type="string">
      Wallet currency (e.g. `NGN`).
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl -X GET "https://api.payvessel.com/pms/wallets" \
  -H "Authorization: Bearer sk_test_..."
```

# Initialize Transaction

**Initialize a payment session** for the Checkout flow. This endpoint creates a temporary session and returns a `checkout_url` where you can redirect your customer to complete their payment.

## Endpoint

<Badge variant="success">**POST**</Badge> `/pms/transactions/initialize`

## Headers

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `Authorization` | string | **Required** | Bearer token with your secret API key |
| `Content-Type` | string | **Required** | Must be `application/json` |

## Request Body

<ParamField body="customer_email" type="string" required>
  The customer's email address.
</ParamField>

<ParamField body="customer_name" type="string" required>
  The customer's full name.
</ParamField>

<ParamField body="customer_phone_number" type="string" required>
  The customer's phone number.
</ParamField>

<ParamField body="amount" type="string" required>
  Amount to charge in the smallest currency unit (e.g., `"10000"` for â‚¦100.00).
</ParamField>

<ParamField body="currency" type="string" default="NGN">
  Currency code for the transaction.
</ParamField>

<ParamField body="channels" type="array">
  List of enabled payment methods. Options: `["BANK_TRANSFER", "CARD"]`.
</ParamField>

<ParamField body="reference" type="string">
  Your unique transaction reference. If not provided, PayVessel will generate one.
</ParamField>

<ParamField body="redirect_url" type="string">
  The URL users should be redirected to after payment.
</ParamField>

<ParamField body="metadata" type="object">
  Additional data you want to store with the transaction.
</ParamField>

## Response

<ResponseField name="status" type="string">
  Request status - `"success"` or `"error"`.
</ResponseField>

<ResponseField name="message" type="string">
  Description of the result.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Data Object">
    <ResponseField name="data.reference" type="string">
      The transaction reference.
    </ResponseField>
    <ResponseField name="data.checkout_url" type="string">
      The URL to redirect the customer to.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl -X POST https://api.payvessel.com/pms/transactions/initialize \
  -H "Authorization: Bearer sk_test_..." \
  -H "Content-Type: application/json" \
  -d '{
    "customer_email": "customer@example.com",
    "customer_name": "John Doe",
    "amount": "500000",
    "currency": "NGN",
    "channels": ["CARD", "BANK_TRANSFER"]
  }'
```

## Example Response

```json
{
  "status": "success",
  "message": "Checkout initialized",
  "data": {
    "reference": "PV_123456789",
    "checkout_url": "https://payvessel.com/checkout/PV_123456789"
  }
}
```
